## Manier van werken:

Indien je denkt dat de andere ook kan werken aan een liedje, communiceer, zodat
er geen conflict is in pushen.

## Format
5 stemmige muziek: Klein 1, 2, Zwaar 1, 2, Bas, (middenstem?)

Mogelijkheid om daarna nog uit te bereiden naar volledige fanfare arangement,
moet geen extra stemmen hebben, gewoon uitgebreid.
